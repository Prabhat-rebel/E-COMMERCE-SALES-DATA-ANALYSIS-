📊 **Sales Data Analysis Project**

📌**Project Overview**
This project focuses on analyzing sales data using Excel and Power BI to generate meaningful insights and interactive dashboards.

🎯 **Objective**
- To analyze sales performance
- To identify trends and patterns
- To create an interactive dashboard for better decision making

🛠️ **Tools Used**
- Microsoft Excel
- Power BI

📂 **Project Files**
- sales_data.xlsx (Dataset)
- sales_dashboard.pbix (Power BI Dashboard)
- Project_Report.pdf (Detailed Report)

📊 **Dashboard Features**
- Sales by Region
- Monthly Sales Trends
- Top Products
- KPI Cards (Total Sales, Profit, Orders)

📈 **Key Insights**
- Highest sales observed in XYZ region
- Peak sales during certain months
- Top-performing products identified

🚀 **Conclusion**
This project demonstrates how data visualization tools like Power BI can transform raw data into actionable insights.
